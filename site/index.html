openrate

Open, ZAR-anchored exchange rates — the open way.

openrate is an open-source exchange-rate engine. It ingests rates the open way — from central-bank reference files and free public venue feeds, not by reselling a paid API — models every currency as a graph rather than a single base, and serves an all-pairs JSON API plus an embedded React UI from one Go binary.

Everything in one binary

Self-hostable, open source, and built to run anywhere.

A graph, not a base

Every source keeps its quotes in their native base as edges in a currency graph. Any pair is the product of rates along the shortest path, so a direct quote always beats a triangulated cross.

Open, ZAR-anchored sourcing

Ingests central-bank files and free public feeds — ECB, SARB, Coinbase, Luno and more — never reselling a paid API. ZAR is the anchor for free: just the default view over the graph.

Quality on every number

Each rate carries a grade A–D plus confidence, combining freshness, directness, source authority, cross-source agreement, and per-currency caveats.

All-pairs JSON API

GET /api/v1/rates, /convert, and /meta — every response includes hops, as_of, age, the resolution path, and the contributing sources.

Interest-rate engine

An optional flat time-series engine covering 49 central banks’ policy rates with daily history, plus the South African ZARONIA family — served alongside the FX API.

Binary or Go library

Serve the API and embedded React UI from a single binary, or import the package and run the engine in-process — same store, sources, and hardening.

Quick start

Up and running in a couple of commands.

go run ./cmd/openrate          # serves :8080, base ZAR, hourly refresh
# or build a single binary
go build -o openrate ./cmd/openrate && ./openrate -addr :8080 -base ZAR

# or with Docker
docker build -t openrate . && docker run -p 8080:8080 openrate
